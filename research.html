Implementation and evaluation of Artificial neural network for Bangladesh.

What amount of power will be used in future periods for Bangladesh that has predicted using Artificial neural network. To implement and evaluate the Artificial neural network, MATLAB software has used and finally, pertial work has been done to found out for next 10(Ten) years from 2013 to 2022.


The use of artificial neural networks (ANN) has been a widely studied load forecasting technique since 1990. Artificial neural networks (ANNs) have succeeded in several power system problems, such as planning, control, analysis, protection, design, load forecasting, security analysis, and fault diagnosis. The last three are the most popular. The ANNs ability in mapping complex non-linear relationships is responsible for the growing number of its application to load forecasting. Most of the ANNs have been applied to short-term load forecasting. Only a few studies are carries out for long-term load demand forecasting. In applying a neural network to load forecasting, one must select one of a number of architectures the number and connectivity of layers and elements, use of bi-directional or uni-directional links and the number format to be used by inputs and outputs.


Neural Network is a mathematical or computational model based on the structure and functional aspects of biological neural networks. It consists of an interconnected group of artificial neurons, and it processes information using a connectionist approach to computation [2]. Neural networks are essential for non-linear circuits that have the demonstrated capability to do non-linear curve fitting. The outputs of neural network are some linear or non-linear mathematical function of its inputs. Network elements are arranged in a relatively small number of connected layers of elements between network Inputs and outputs. Feedback paths are also sometimes used. In neural networks model three types of variables are used as inputs: seasonal related inputs, weather related inputs, and historical loads. In applying a neural network to load forecasting, one must select one of a number of architectures, the number and connectivity of layers and elements, use of bi-directional or uni-directional links and the number format to be used by inputs and outputs [1]. For short-term load forecasting based on neutral network model three-layer feed forward and a back propagation algorithm is used for training.
